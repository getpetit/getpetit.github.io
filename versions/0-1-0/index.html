v0.1.0

  • Completely rewritten from Electron to native Swift/AppKit. Lighter and faster.
  • PiP improved with larger controls, higher quality artwork, and artist name display.
  • Media keys (play/pause, next, previous) are now supported.
  • Shuffle/Repeat icons in menubar now show active/inactive state.
  • Wide Mode added. Expand the player window beyond default width from settings.
  • Keyboard shortcuts changed to Ctrl+Option combos to avoid conflicts with other apps.
  • Added "Import Login from Chrome" in right-click menu for when Google sign-in is blocked in the embedded browser.
  • 9 languages supported: English, Korean, Japanese, Simplified Chinese, Spanish, German, Portuguese, French, and Turkish.
  • AdBlock removed.
  • Track change notifications removed.
