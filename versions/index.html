Release Notes

v0.1.2 2026-04-06

  • Fixed an issue where the settings menu appeared behind the app window when clicking the gear icon.

v0.1.1 2026-04-05

  • Keyboard shortcuts can now be customized. Go to Settings → Keyboard Shortcuts → Customize Shortcuts...

v0.1.0 2026-04-04

  • Completely rewritten from Electron to native Swift/AppKit. Lighter and faster.
  • PiP improved with larger controls, higher quality artwork, and artist name display.
  • Media keys (play/pause, next, previous) are now supported.
  • Shuffle/Repeat icons in menubar now show active/inactive state.
  • Wide Mode added. Expand the player window beyond default width from settings.
  • Keyboard shortcuts changed to Ctrl+Option combos to avoid conflicts with other apps.
  • Added "Import Login from Chrome" in right-click menu for when Google sign-in is blocked in the embedded browser.
  • 9 languages supported: English, Korean, Japanese, Simplified Chinese, Spanish, German, Portuguese, French, and Turkish.
  • Track change notifications removed.

v0.0.9 2020-03-30

  • Relaunch re-supported
  • Minor bug fix

v0.0.8 2020-03-29

  • Auto Updater minor fix

v0.0.7 2020-03-28

  • Support Auto Update

v0.0.6 2020-03-27

  • Support changing play time in "now playing" (also in sidebar).
  • Support notification when music changes.
  • Support repeat status changing.
  • Fix a bug occurred by dragging.

v0.0.5 2020-03-20

  • Offline mode supported.
  • Bug fix.

v0.0.4 2020-03-15

  • Picture in Picture (PIP) is supported.
  • Keyboard shortcuts are supported (Toggle window, Toggle Play & Stop, Play previous song, Play next song)
  • Added direct buttons to play / pause in the menu bar, play previous song, play next song.
  • English is supported.
  • Bug fixes and UI improvements.

v0.0.3 2020-02-13

  • "Launch at login" is supported.
